• This site is for the Science Olympiad Division C Astronomy 2018-2019 event
    • It is designed to work offline, without internet connectivity
  • To navigate through the different pages of this site, use the header (desktop) or drawer (mobile)
    • The page you are currently on is bolded
  • On the header, there are 2 buttons: SHOW and HIDE
    • The SHOW button displays an overlay of content on top of the current page
      • Home: the rules for the Astronomy event are displayed
      • DSOs: Google Images search results are displayed
    • The HIDE button removes the content overlay
